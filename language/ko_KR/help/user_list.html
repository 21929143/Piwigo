사용자 목록

Piwigo 갤러리의 사용자를 관리하는 곳입니다.

사용자 추가

관리자는 사용자를 수동으로 추가 할 수 있습니다. 각 사용자에 대해 사용자 이름, 비밀번호를 생성하고 유효한 이메일 주소를 제출합니다.

사용자 목록

기존 사용자 목록은 사용자 이름 (*를 와일드 카드로 사용), 그룹 또는 상태로 필터링 할 수 있습니다. 등록 날짜 또는 사용자 이름별로 오름차순 또는 내림차순으로 정렬 할 수 있습니다.

이 화면에서는 여러 작업을 사용하여 한 번에 여러 사용자를 관리 할 수 있습니다:

  • 사용자 삭제 (확인 필요)
  • 사용자 상태 변경
  • 그룹에서 연결 또는 분리
  • 표시 속성 수정
  • 추가 기본설정 수정/li>

대상은 선택한 사용자 (기본값) 또는 필터링 된 목록에 표시된 모든 사용자입니다.
